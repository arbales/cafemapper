    mixin CafeMapper.Modules.ServerCommunication

    Description

    CafeMapper.Modules.ServerCommunication provides a set of instance methods to CafeMapper.DataObjects that enable communication with a server-side data store.

    Include this module not with DataObject#addMethods directly, but by using CafeMapper.DataObject.include, which is available on all DataObject classes.

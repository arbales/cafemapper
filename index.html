CafeMapper is a simple localStorage ORM.

It can synchronize with server storage.

A la carte.

var Person = $MODEL("Person");
var p = new Person({id: 4, name: 'Jon Smith', zipcode: 20646});
p.store();
d = Person.get(4);

Binds data objects with simple eventing to keep your displayed elements in-sync with your local and remote data stores. DataMapper in mind.

CafeMapper isn't finished yet, and the API will change constantly. Don't use CafeMapper (at all).

The source for CafeMapper is at github.com/arbales/cafemapper.
